# ILYB Terms of Use

This repository contains the web page for the ILYB app Terms & Conditions.

## Publish to GitHub Pages

1. Push this folder's contents to the repository:  
   `https://github.com/hwsw9sqvg2-create/ILYB-Terms-of-Use`
2. In the repo: **Settings → Pages**
3. Under **Source**, choose **Deploy from a branch**
4. Branch: **main** (or **master**), folder: **/ (root)**
5. Save. The site will be available at:  
   `https://hwsw9sqvg2-create.github.io/ILYB-Terms-of-Use/`

## Language

- Default language: **English**
- Use the language switcher at the top to change language
- The app can open the page with `?lang=uk`, `?lang=ru`, etc. to show the matching language
